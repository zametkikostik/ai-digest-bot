## ✅ ЗАМЕНА ПЛАТНЫХ API НА БЕСПЛАТНЫЕ

### 1️⃣ Yandex SpeechKit → gTTS (Google TTS)

**Было:**
```python
# Yandex SpeechKit (требует API ключ)
YANDEX_API_KEY=...
YANDEX_FOLDER_ID=...
```

**Стало:**
```python
# gTTS (Google Text-to-Speech) - БЕСПЛАТНО!
# Не требует API ключей!
pip install gtts
```

**Преимущества:**
- ✅ Полностью бесплатно
- ✅ Не требует API ключей
- ✅ 40+ языков (включая русский)
- ✅ Хорошее качество голоса
- ✅ Простая интеграция

**Поддерживаемые языки:**
```

🇫🇷 Français
🇪🇸 Español
🇮🇹 Italiano
🇵🇹 Português
🇵🇱 Polski
🇺🇦 Українська
🇧🇾 Беларуская
🇰🇿 Қазақша
🇺🇿 O'zbek
🇹🇷 Türkçe
🇸🇦 العربية
🇮🇱 עברית
🇨🇳 中文
🇯🇵 日本語
🇰🇷 한국어
```

**Использование:**
```python
from core.free_tts import free_tts, send_voice_message

# Генерация речи
audio_data = await free_tts.text_to_speech(
    text="Привет! Я AIден!",
    lang="ru"
)

# Отправка в Telegram
await send_voice_message(
    bot=bot,
    chat_id=chat_id,
    text="Привет! Я AIден!",
    lang="ru"
)
```

**Ограничения:**
- Макс. длина текста: ~200 символов за запрос
- Скорость: нормальная или медленная
- 1 голос (женский по умолчанию)

```python
async def cmd_weather(message: types.Message):

    city = "Москва"  # Из аргументов

    )

```

## ⚠️ ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ

### gTTS (Google TTS)

**Минусы:**
- 1 голос (женский)
- Нет эмоций
- Нет регулировки скорости (только slow=True)
- Требует интернет
- Может блокироваться при частых запросах

**Решение:**
- Делать задержки между запросами
- Кэшировать часто используемые фразы
- Использовать для коротких сообщений

### Open-Meteo

**Минусы:**
- Нет точной влажности
- Нет feels_like
- Меньше городов чем у OpenWeather
- Нет исторических данных

**Решение:**
- Использовать примерные значения
- Добавлять дисклеймер
- Для точных данных — платный API
