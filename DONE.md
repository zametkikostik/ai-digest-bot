## 🚀 Как использовать

### Для пользователей:
1. Нажмите `/faq` или кнопку "📚 Категории вопросов"
2. Выберите категорию
3. Нажмите на вопрос для просмотра ответа

### Для администраторов:

#### Быстрый старт:
```bash
# 1. Заполните тестовыми данными
python fill_faq_data.py

# 2. Проверьте в боте
/faq
```

#### Добавление своей категории:
```
/addcategory 🤖 Нейросети|Всё о нейронных сетях|🤖
```

#### Добавление вопроса:
```
/addquestion 1|Что такое трансформер?|Трансформер — это архитектура...|трансформер, attention, nlp
```

#### Просмотр статистики:
```
/faqs
/listcategories
```
