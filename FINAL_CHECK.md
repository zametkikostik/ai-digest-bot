## ☁️ CLOUDFLARE vs ЛОКАЛЬНО

### ❌ Почему бот не в Cloudflare?

**Текущий статус Webhook:**
```json
{
    "url": "",  ← ПУСТОЙ!
    "has_custom_certificate": false
}
```

**Причина:** Бот настроен на **локальный запуск** (polling), а не Cloudflare (webhook).

### ✅ Почему локальный запуск — это хорошо?

| Функция | Cloudflare | Локально |
|---------|------------|----------|
| RAG | Упрощённый (KV) | Полный (ChromaDB) ✅ |
| Планировщик | Cron (платно) | APScheduler ✅ |
| Сложность | Средняя | Низкая ✅ |
| Кнопки | Работают | Работают ✅ |
| Premium | Работает | Работает ✅ |

**Вывод:** Локальный запуск обеспечивает **полный функционал** из ADMIN_STATS.md!

## 🔧如果想 деплой на Cloudflare

### Инструкция: `CLOUDFLARE_DEPLOY_FIX.md`

```bash
cd cloudflare
wrangler login
wrangler secret put BOT_TOKEN
wrangler secret put OPENROUTER_API_KEY
wrangler deploy
```

После деплоя:
```bash
curl -X POST "https://api.telegram.org/bot<TOKEN>/setWebhook" \
    -d "url=https://ai-digest-bot.<subdomain>.workers.dev"
```
