## 🚀 ИСПОЛЬЗОВАНИЕ

### Голосовые сообщения

```python
from core.free_tts import send_voice_message

@router.message(Command("voice"))
async def cmd_voice(message: types.Message):
    await send_voice_message(
        bot=message.bot,
        chat_id=message.chat.id,
        text="Привет! Я AIден!",
        lang="ru"
    )
```

### Погода

```python
from core.free_weather import free_weather

@router.message(Command("weather"))
async def cmd_weather(message: types.Message):
    city = "Москва"
    weather = await free_weather.get_weather(city)
    
    text = (
        f"🌤️ **Погода: {weather['city']}**\n"
        f"🌡️ {weather['temp']}°C, {weather['description']}\n"
        f"💨 Ветер: {weather['wind_speed']} м/с"
    )
    
    await message.reply(text, parse_mode='Markdown')
```
