## Основы криптовалют

### Что такое криптовалюта

**Криптовалюта** — цифровая валюта, использующая криптографию для безопасности и работающая на технологии блокчейн.

**Ключевые особенности:**
- 🔐 **Децентрализация** — нет единого центра управления
- 🔒 **Безопасность** — криптографическая защита
- 📝 **Прозрачность** — все транзакции публичны
- ⚡ **Быстрые переводы** — минуты вместо дней
- 🌍 **Глобальность** — доступна везде
- 💰 **Псевдоанонимность** — не требует личных данных

### История криптовалют

**2008:** Сатоши Накамото публикует whitepaper Биткоина
**2009:** Запуск сети Биткоин, первый майнинг
**2010:** Первая покупка за биткоины (2 пиццы за 10 000 BTC)
**2011:** Появление альткоинов (Litecoin, Namecoin)
**2013:** Виталик Бутерин предлагает Ethereum
**2015:** Запуск Ethereum, смарт-контракты
**2017:** Бум ICO, биткоин достигает $20 000
**2020:** DeFi лето, рост децентрализованных финансов
**2021:** NFT бум, биткоин ATH $69 000
**2022:** Криптозима, крах FTX, Terra/Luna
**2023:** Восстановление, одобрение Bitcoin ETF
**2024:** Bitcoin ATH $73 000, рост институционального принятия

## Блокчейн

### Что такое блокчейн

**Блокчейн** — распределённая база данных, хранящая информацию в виде цепочки блоков.

**Структура блока:**
```
┌─────────────────────────────────┐
│  Заголовок блока                │
│  ├─ Хэш предыдущего блока       │
│  ├─ Хэш текущего блока          │
│  ├─ Временная метка             │
│  ├─ Nonce (случайное число)     │
│  └─ Merkle Root (корень дерева) │
├─────────────────────────────────┤
│  Транзакции                     │
│  ├─ Транзакция 1                │
│  ├─ Транзакция 2                │
│  └─ ...                         │
└─────────────────────────────────┘
```

### Принципы работы

1. **Транзакция создаётся** — пользователь инициирует перевод
2. **Транзакция подписывается** — приватным ключом
3. **Транзакция передаётся в сеть** — нодам
4. **Майнеры/валидаторы проверяют** — корректность
5. **Транзакция включается в блок** — формирование блока
6. **Блок добавляется в цепочку** — консенсус
7. **Подтверждение** — транзакция завершена

### Типы блокчейнов

#### Публичные (Public)
- Открыты для всех
- Полная децентрализация
- Примеры: Bitcoin, Ethereum, Solana

#### Приватные (Private)
- Доступ по разрешению
- Контролируются организацией
- Примеры: Hyperledger, Corda

#### Гибридные (Consortium)
- Несколько организаций управляют
- Частичная децентрализация
- Примеры: R3 CEV, B3i

### Механизмы консенсуса

#### Proof of Work (PoW)
**Как работает:**
- Майнеры решают сложные математические задачи
- Первый нашедший решение получает награду
- Требует много энергии

**Плюсы:**
- ✅ Высокая безопасность
- ✅ Проверен временем
- ✅ Истинная децентрализация

**Минусы:**
- ❌ Высокое энергопотребление
- ❌ Низкая пропускная способность
- ❌ Дорогой вход для майнеров

**Примеры:** Bitcoin, Litecoin, Dogecoin, Kaspa

#### Proof of Stake (PoS)
**Как работает:**
- Валидаторы блокируют монеты (стейкинг)
- Выбор валидатора пропорционален ставке
- Награда за валидацию блоков

**Плюсы:**
- ✅ Энергоэффективность
- ✅ Высокая пропускная способность
- ✅ Дешевле участие

**Минусы:**
- ❌ Богатые становятся богаче
- ❌ Меньше децентрализации
- ❌ Атака 51% дешевле

**Примеры:** Ethereum 2.0, Cardano, Solana, Polkadot

#### Другие механизмы
- **DPoS** (Delegated PoS) — голосование за делегатов (EOS, TRON)
- **PoH** (Proof of History) — доказательство времени (Solana)
- **PoA** (Proof of Authority) — авторитетные валидаторы
- **PoSpace** — доказательство места (Chia)

**2021:** ATH $69 000
**2024:** ATH $73 000, одобрение spot ETF

### Технические характеристики

| Параметр | Значение |
|----------|----------|
| Алгоритм | SHA-256 |
| Время блока | ~10 минут |
| Максимум монет | 21 000 000 BTC |

| TPS | ~7 транзакций в секунду |

### Халвинг

**Что это:** Уменьшение награды майнерам в 2 раза

**История халвингов:**
- 2009: 50 BTC
- 2012: 25 BTC
- 2016: 12.5 BTC
- 2020: 6.25 BTC
- 2024: 3.125 BTC
- ~2028: 1.5625 BTC
- ~2140: Последняя сатоши

**Влияние на цену:**
- Исторически после халвинга следовал рост
- Уменьшение предложения при растущем спросе
- Не гарантия будущего роста

### Bitcoin ETF

**Что это:** Биржевой фонд, отслеживающий цену биткоина

**Одобрены в США (январь 2024):**
- BlackRock iShares Bitcoin Trust (IBIT)
- Fidelity Wise Origin Bitcoin Fund (FBTC)
- Grayscale Bitcoin Trust (GBTC)
- Ark 21Shares Bitcoin ETF (ARKB)
- И другие

**Значение:**
- Институциональный доступ
- Упрощённое инвестирование
- Легитимизация актива

### Lightning Network

**Проблема:** Низкая пропускная способность Bitcoin

**Решение:** Lightning Network — сеть второго уровня

**Как работает:**
- Платёжные каналы между пользователями
- Мгновенные транзакции вне цепочки
- Фиксация итогового состояния в блокчейне

**Преимущества:**
- ⚡ Мгновенные платежи
- 💸 Минимальные комиссии
- 📈 Миллионы TPS теоретически

## Эфириум

### Что такое Ethereum

**Ethereum** — децентрализованная платформа для смарт-контрактов и dApps.

**Отличия от Биткоина:**
- Смарт-контракты (программируемые)
- Виртуальная машина (EVM)
- Газ (комиссия за вычисления)
- Быстрее блоки (~12 секунд)

### Смарт-контракты

**Что это:** Программы, выполняющиеся на блокчейне.

**Пример простого контракта:**
```solidity
// Простой контракт хранения
contract SimpleStorage {
    uint256 private data;
    
    function set(uint256 _data) public {
        data = _data;
    }
    
    function get() public view returns (uint256) {
        return data;
    }
}
```

**Применение:**
- DeFi протоколы
- NFT маркетплейсы
- DAO (децентрализованные организации)
- Токенизация активов
- Игры (GameFi)

### Газ (Gas)

**Что это:** Единица измерения вычислительной работы

**Формула комиссии:**
```
Комиссия = Gas Used × Gas Price
```

**Пример:**
- Простой перевод: 21 000 gas
- Swap на Uniswap: ~150 000 gas
- Минт NFT: ~200 000 gas

**Gas Price:** Зависит от загруженности сети (Gwei)

### Ethereum 2.0 (The Merge)

**Переход на PoS:** Сентябрь 2022

**Улучшения:**
- ✅ Энергопотребление -99.95%
- ✅ Выпуск ETH уменьшился
- ✅ Основа для шардинга

**Будущие обновления:**
- Шардинг (горизонтальное масштабирование)
- Proto-danksharding (EIP-4844)
- Улучшение масштабируемости

### Стейкинг ETH

**Требования:**
- Минимум 32 ETH для валидатора
- Или через пулы (Lido, Rocket Pool)

**Доходность:** 3-5% годовых в ETH

**Риски:**
- Слэшинг (штраф за плохую валидацию)
- Блокировка до выхода (после Shanghai)

## Торговля и инвестирование

### Стратегии инвестирования

#### HODL
**Купить и держать:**
- Долгосрочная стратегия
- Игнорирование волатильности
- Подходит для BTC, ETH

#### DCA (Dollar Cost Averaging)
**Регулярные покупки:**
- Покупка на фиксированную сумму
- Регулярно (еженедельно/ежемесячно)
- Усреднение цены входа

**Пример:**
```
Ежемесячно: $1000
Срок: 12 месяцев
Результат: Усреднённая цена
```

#### Трейдинг
**Активная торговля:**

**Дневная торговля:**
- Открытие и закрытие в один день
- Использование волатильности
- Высокий риск

**Свинг-трейдинг:**
- Удержание позиций дни/недели
- Ловля трендов
- Меньше стресса

### Технический анализ

**Основные индикаторы:**
- MA (Moving Average) — скользящая средняя
- RSI (Relative Strength Index) — перекупленность/перепроданность
- MACD — трендовый индикатор
- Bollinger Bands — волатильность
- Volume — объём торгов

**Уровни:**
- Поддержка — цена, где покупают
- Сопротивление — цена, где продают
- Пробой — преодоление уровня

3. Никогда не храните в цифровом виде
4. Никогда не делитесь ни с кем
5. Используйте парольную фразу (25-е слово)

**Операционная безопасность:**
1. Отдельный браузер для крипто
2. Блокировщики рекламы (uBlock)
3. Антивирус
4. 2FA на всех аккаунтах (не SMS!)
5. Проверяйте контракты перед взаимодействием

**Резервное копирование:**
- Несколько копий сид-фразы

- Наследственный план
