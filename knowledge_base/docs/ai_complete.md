## Что такое ИИ

**Искусственный интеллект (ИИ, Artificial Intelligence, AI)** — способность компьютерных систем выполнять задачи, требующие человеческого интеллекта: обучение, понимание языка, распознавание образов, принятие решений, творчество.

### Ключевые характеристики:
- **Обучение** — способность улучшать производительность на основе опыта
- **Адаптация** — подстройка под новые условия
- **Автономность** — работа без постоянного контроля человека
- **Понимание языка** — обработка естественного языка (NLP)
- **Распознавание образов** — компьютерное зрение, анализ изображений

## История развития

### 1950-е: Зарождение
- **1950** — Алан Тьюринг публикует статью "Вычислительные машины и разум", предлагает тест Тьюринга
- **1956** — Джон Маккарти вводит термин "искусственный интеллект" на конференции в Дартмуте
- **1957** — Фрэнк Розенблатт создаёт перцептрон

### 1960-1970-е: Первые успехи
- **1966** — ELIZA, первый чат-бот
- **1972** — Prolog, язык логического программирования
- **1974** — Терри Виноград создаёт SHRDLU

### 1980-1990-е: Экспертные системы
- **1980-е** — Бум экспертных систем
- **1997** — Deep Blue побеждает чемпиона мира по шахматам Гарри Каспарова

### 2000-е: Машинное обучение
- **2006** — Джеффри Хинтон вводит термин "глубокое обучение"
- **2012** — AlexNet побеждает в ImageNet, революция в компьютерном зрении

### 2010-е: Глубокое обучение
- **2016** — AlphaGo побеждает чемпиона мира по го
- **2017** — Трансформеры (Attention Is All You Need)
- **2018** — BERT, GPT

### 2020-е: Большие языковые модели
- **2020** — GPT-3 (175 млрд параметров)
- **2022** — ChatGPT запускается для публики
- **2023** — GPT-4, мультимодальные модели
- **2024** — GPT-4o, Claude, Gemini, Llama

## Виды ИИ

### По уровню возможностей

#### 1. Слабый ИИ (Narrow AI, ANI)
Решает конкретные задачи в ограниченной области:
- Siri, Alexa, Google Assistant
- Рекомендательные системы (Netflix, YouTube)
- Распознавание лиц
- Автопилот Tesla
- Шахматные программы

**Характеристики:**
- ✅ Превосходит человека в своей области
- ❌ Не может работать вне своей специализации

#### 2. Сильный ИИ (Artificial General Intelligence, AGI)
Универсальный интеллект, способный решать любые интеллектуальные задачи:
- Пока не создан
- Может обучаться любым навыкам
- Обладает сознанием и самосознанием

**Прогнозы создания:** 2030-2060 гг.

#### 3. Суперинтеллект (Artificial Superintelligence, ASI)
Интеллект, превосходящий человеческий во всех областях:
- Гипотетическая концепция
- Может привести к технологической сингулярности

## Машинное обучение

**Машинное обучение (Machine Learning, ML)** — подраздел ИИ, создающий алгоритмы, которые обучаются на данных.

### Типы обучения

#### 1. Обучение с учителем (Supervised Learning)
Алгоритм обучается на размеченных данных (вход → правильный ответ):

**Задачи:**
- **Классификация** — отнесение к категории (спам/не спам)
- **Регрессия** — предсказание числа (цена дома)

**Алгоритмы:**
- Линейная регрессия
- Логистическая регрессия
- Метод опорных векторов (SVM)
- Решающие деревья
- Случайный лес
- Градиентный бустинг (XGBoost, LightGBM, CatBoost)

#### 2. Обучение без учителя (Unsupervised Learning)
Алгоритм ищет паттерны в данных без меток:

**Задачи:**
- **Кластеризация** — группировка объектов
- **Снижение размерности** — упрощение данных
- **Поиск аномалий** — обнаружение выбросов

**Алгоритмы:**
- K-means
- Иерархическая кластеризация
- DBSCAN
- PCA (Principal Component Analysis)
- Автоэнкодеры

#### 3. Обучение с подкреплением (Reinforcement Learning)
Алгоритм обучается через взаимодействие со средой, получая награды/штрафы:

**Применение:**
- Игры (AlphaGo, AlphaStar)
- Робототехника
- Автономные системы
- Торговые боты

**Ключевые концепции:**
- Агент (agent)
- Среда (environment)
- Действие (action)
- Награда (reward)
- Политика (policy)

## Глубокое обучение

**Глубокое обучение (Deep Learning, DL)** — подраздел ML, использующий многослойные нейронные сети.

### Архитектуры нейронных сетей

#### 1. Полносвязные сети (Fully Connected, FC)
Каждый нейрон связан со всеми нейронами следующего слоя.

**Применение:**
- Табличные данные
- Простые задачи классификации

#### 2. Сверточные сети (Convolutional Neural Networks, CNN)
Используют свёртки для обработки изображений.

**Слои:**
- Convolutional — извлечение признаков
- Pooling — уменьшение размерности
- Fully Connected — классификация

**Применение:**
- Распознавание изображений
- Обработка видео
- Медицинская диагностика
- Автопилоты

**Известные архитектуры:**
- LeNet (1998)
- AlexNet (2012)
- VGG (2014)
- ResNet (2015)
- EfficientNet (2019)

#### 3. Рекуррентные сети (Recurrent Neural Networks, RNN)
Обрабатывают последовательности, имеют "память".

**Применение:**
- Обработка текста
- Распознавание речи
- Прогнозирование временных рядов

**Варианты:**
- LSTM (Long Short-Term Memory)
- GRU (Gated Recurrent Unit)

#### 4. Трансформеры (Transformers)
Архитектура на основе механизма внимания (attention).

**Преимущества:**
- Параллельная обработка
- Долгосрочные зависимости
- Масштабируемость

**Применение:**
- NLP (перевод, генерация текста)
- Компьютерное зрение (ViT)
- Мультимодальные задачи

## Нейронные сети

### Структура нейрона

```
Входы (x1, x2, ..., xn)
    ↓
Веса (w1, w2, ..., wn)
    ↓
Сумматор: z = Σ(wi * xi) + b
    ↓
Функция активации: a = f(z)
    ↓
Выход
```

### Функции активации

| Функция | Формула | Применение |
|---------|---------|------------|
| Sigmoid | 1/(1+e^-x) | Бинарная классификация |
| Tanh | (e^x - e^-x)/(e^x + e^-x) | Скрытые слои |
| ReLU | max(0, x) | Глубокие сети |
| Leaky ReLU | max(0.01x, x) | Устраняет "мёртвые" нейроны |
| Softmax | e^xi / Σe^xj | Многоклассовая классификация |

### Процесс обучения

1. **Инициализация весов** — случайные значения
2. **Прямое распространение** — вычисление выхода
3. **Вычисление ошибки** — сравнение с правильным ответом
4. **Обратное распространение** — вычисление градиентов
5. **Обновление весов** — градиентный спуск
6. **Повтор** — до сходимости

### Оптимизаторы

- **SGD** — стохастический градиентный спуск
- **Momentum** — добавляет инерцию
- **Adam** — адаптивная оценка моментов (самый популярный)
- **RMSprop** — адаптивная скорость обучения

## Трансформеры и GPT

### Архитектура трансформера

**Ключевые компоненты:**

1. **Self-Attention** — внимание к разным частям входа
2. **Multi-Head Attention** — несколько механизмов внимания
3. **Positional Encoding** — кодирование позиции токенов
4. **Feed-Forward Networks** — обработка после внимания
5. **Layer Normalization** — нормализация
6. **Residual Connections** — остаточные связи

### GPT (Generative Pre-trained Transformer)

**Поколения:**

| Модель | Год | Параметры | Особенности |
|--------|-----|-----------|-------------|
| GPT | 2018 | 117 млн | Первая версия |
| GPT-2 | 2019 | 1.5 млрд | Улучшенная генерация |
| GPT-3 | 2020 | 175 млрд | Few-shot learning |
| GPT-3.5 | 2022 | ~175 млрд | Оптимизация для диалога |
| GPT-4 | 2023 | ~1 трлн | Мультимодальность |
| GPT-4o | 2024 | ~1 трлн | Omni (текст, аудио, видео) |

### Принцип работы GPT

1. **Токенизация** — разбиение текста на токены
2. **Embedding** — векторное представление токенов
3. **Positional Encoding** — добавление информации о позиции
4. **Transformer Blocks** — обработка через слои внимания
5. **Output** — предсказание следующего токена

### Prompt Engineering

**Техники:**

- **Zero-shot** — выполнение задачи без примеров
- **Few-shot** — несколько примеров в промпте
- **Chain of Thought** — пошаговое рассуждение
- **Role prompting** — задание роли ("ты эксперт...")
- **System prompts** — системные инструкции

## Этика и безопасность

### Проблемы

#### 1. Предвзятость (Bias)
- ИИ наследует предвзятость из данных
- Дискриминация по расе, полу, возрасту
- Несправедливые решения в кредитовании, найме

**Решения:**
- Сбалансированные датасеты
- Аудит алгоритмов
- Разнообразие в командах разработчиков

#### 2. Конфиденциальность
- Сбор персональных данных
- Распознавание лиц в общественных местах
- Массовая слежка

**Решения:**
- Privacy-preserving ML
- Федеративное обучение
- Дифференциальная приватность

#### 3. Замена рабочих мест
- Автоматизация рутинных задач
- Исчезновение профессий
- Социальное неравенство

**Решения:**
- Переобучение работников
- Универсальный базовый доход
- Новые профессии в сфере ИИ

#### 4. Безопасность
- Злонамеренное использование ИИ
- Deepfakes
- Автономное оружие
- Взлом ИИ-систем

**Решения:**
- Регулирование
- Этические принципы
- Технические защиты

### Принципы ответственного ИИ

1. **Прозрачность** — объяснимость решений
2. **Справедливость** — отсутствие дискриминации
3. **Подотчётность** — ответственность за решения
4. **Безопасность** — защита от злоупотреблений
5. **Конфиденциальность** — уважение приватности
6. **Человекоцентричность** — ИИ на службе людям
