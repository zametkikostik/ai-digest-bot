# 📚 Руководство по системе категорий и FAQ

## Описание

Бот теперь поддерживает полноценную систему категорий с вопросами и ответами. Пользователи могут просматривать вопросы по категориям, искать по ключевым словам и получать ответы.

---

## 📋 Команды для пользователей

| Команда | Описание |
|---------|----------|
| `/faq` | Показать все категории вопросов |
| `/categories` | Алиас для `/faq` |

---

## 🔧 Команды для администраторов

### Управление категориями

#### `/addcategory Название|Описание|Emoji`
Создать новую категорию.

**Пример:**
```
/addcategory 🤖 Основы AI|Базовые понятия об искусственном интеллекте|🤖
```

#### `/editcategory ID|Название|Описание|Emoji`
Редактировать существующую категорию.

**Пример:**
```
/editcategory 1|🤖 Основы Искусственного Интеллекта|Обновлённое описание|🤖
```

#### `/deletecategory ID`
Удалить категорию (все вопросы в ней также будут удалены).

**Пример:**
```
/deletecategory 1
```

#### `/listcategories`
Показать все категории с количеством вопросов.

---

### Управление вопросами

#### `/addquestion CategoryID|Вопрос|Ответ|Ключевые слова`
Добавить новый вопрос-ответ в категорию.

**Пример:**
```
/addquestion 1|Что такое нейросеть?|Нейронная сеть это...|нейросеть, ai, определение
```

#### `/editquestion ID|Вопрос|Ответ|Ключевые слова`
Редактировать существующий вопрос.

**Пример:**
```
/editquestion 1|Что такое нейросеть?|Обновлённый ответ|нейросеть, обновлённые ключевые слова
```

#### `/deletequestion ID`
Удалить вопрос-ответ.

**Пример:**
```
/deletequestion 1
```

#### `/listquestions CategoryID`
Показать все вопросы в категории.

**Пример:**
```
/listquestions 1
```

## 🚀 Быстрый старт

### 1. Заполните тестовыми данными (опционально)

```bash
python fill_faq_data.py
```

Это создаст 5 категорий с 13 вопросами для AI-тематики.

### 2. Добавьте свои категории

```
/addcategory 🤖 Основы AI|Базовые понятия|🤖
/addcategory 💻 Инструменты|AI-инструменты и сервисы|💻
/addcategory 📚 Обучение|Ресурсы для изучения|📚
```

### 3. Добавьте вопросы

```
/addquestion 1|Что такое искусственный интеллект?|ИИ — это способность компьютеров...|ai, определение
/addquestion 2|Какие есть AI-инструменты?|ChatGPT, Midjourney, GitHub Copilot...|инструменты, сервисы
```

### 4. Проверьте

```
/listcategories
/faq
```

## 💡 Лучшие практики

### Структура категорий

1. **Используйте эмодзи** — делает навигацию визуально понятнее
2. **Краткие описания** — помогают пользователям выбрать нужную категорию
3. **Логическая группировка** — объединяйте связанные вопросы

### Написание вопросов

1. **Формулируйте четко** — вопрос должен быть понятен без контекста
2. **Добавляйте ключевые слова** — улучшают поиск
3. **Структурируйте ответы** — используйте списки и форматирование
4. **Обновляйте регулярно** — добавляйте новые вопросы по мере поступления

### Пример хорошего вопроса

```
/addquestion 1|
Как выбрать модель машинного обучения?|
Выбор модели зависит от:

1. **Типа задачи**:
   - Классификация → Logistic Regression, Random Forest
   - Регрессия → Linear Regression, XGBoost
   - Кластеризация → K-Means, DBSCAN

2. **Объёма данных**:
   - Мало данных → простые модели
   - Много данных → нейросети

3. **Требований к скорости**:
   - Быстрое предсказание → лёгкие модели
   - Точность важнее → ансамбли

|машинное обучение, модели, выбор, ml
```

## 📊 Мониторинг

## 🔍 Поиск

Пользователи могут искать вопросы по:
- Тексту вопроса
- Ключевым словам
- Тексту ответа

Поиск активируется кнопкой "🔍 Поиск по вопросам" в меню FAQ.

## 🛠 Техническая информация

### Структура базы данных

**Таблица `categories`:**
- `id` — уникальный идентификатор
- `name` — название категории
- `description` — описание
- `emoji` — эмодзи категории
- `sort_order` — порядок сортировки
- `is_active` — активна ли категория

**Таблица `faq_questions`:**
- `id` — уникальный идентификатор
- `category_id` — ссылка на категорию
- `question` — текст вопроса
- `answer` — текст ответа
- `keywords` — ключевые слова (через запятую)
- `sort_order` — порядок сортировки
- `is_active` — активен ли вопрос
- `views_count` — счетчик просмотров

## 🆘 Troubleshooting

### Категория не отображается
- Проверьте, что `is_active = True`
- Убедитесь, что категория создана в БД

### Вопросы не находятся в поиске
- Добавьте ключевые слова
- Проверьте, что вопрос активен

### Ошибка при добавлении
- Проверьте формат: все параметры через `|`
- Убедитесь, что категория существует
